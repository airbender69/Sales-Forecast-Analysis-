Seasonal Product Sales Forecasting

Project Overview

This project focuses on forecasting seasonal product sales using machine learning techniques, specifically Linear Regression and Random Forest models. The dataset from the Kaggle competition 'Store Sales - Time Series Forecasting' was utilized for model training and evaluation.
